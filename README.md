# Distributed Wallet Ledger

A backend wallet platform built using Java 17, Spring Boot, and PostgreSQL.

## Features Implemented

* Create User
* Create Wallet
* Credit Money
* Debit Money
* Transaction Ledger
* PostgreSQL Persistence
* Spring Data JPA

## APIs

### User APIs

POST /users

### Wallet APIs

POST /wallets

POST /wallets/credit

POST /wallets/debit

## Tech Stack

* Java 17
* Spring Boot 3
* PostgreSQL
* Maven
* JPA/Hibernate
